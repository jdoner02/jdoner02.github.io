The Classroom

CSCD 210: Programming Principles I

This is the class I teachâ€”the first Java course in the CS major. You'll come in with some programming experience (CSCD 110, AP CS, or equivalent) and leave with a solid foundation in object-oriented thinking. We build things from scratch and spend a lot of time on the stuff that trips people up: scope, references, memory, and debugging strategy.

My teaching philosophy: I'd rather you understand why something works than memorize that it works. We'll write buggy code on purpose, trace through memory, and talk about the mental models that make programming click.

I keep Canvas lean for quick reference. For deeper explanations and week-by-week breakdowns, check out the CSCD 210 Wiki in the Skill Treeâ€”it has concept maps, mastery tracking, and the sprawling deep dives that don't fit in a 50-minute lecture.
